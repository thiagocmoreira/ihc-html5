O que é HTML?

  • Hypertext Markup Language

  • Linguagem de Marcação
  • Junção entre os padrões HyTime e SGML
  • Primeira versão feita em 1991 por Tim Berners-Lee
  • Criada para resolver um problema pessoal de Tim
  • Se popularizou junto com a internet
  • Hoje é mantida pela W3C
  • Tornou-se uma norma internacional
