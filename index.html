O que é HTML?

  • Hypertext Markup Language

  • Linguagem de Marcação
  • Junção entre os padrões HyTime e SGML
  • Primeira versão feita em 1991 por Tim Berners-Lee
  • Criada para resolver um problema pessoal de Tim
  • Se popularizou junto com a internet
  • Hoje é mantida pela W3C
  • Tornou-se uma norma internacional

Novidades no HTML5

Faz o que antes só era possível com outras ferramentas e linguagens de programação

Tem suporte à áudio e vídeo de alto nível

  • Suporte a SVG, canvas e outros gráficos vetoriais
  • API interna para rodar scripts
  • Cache de aplicativos e não apenas do navegador
  • Vários elementos adicionados
  • Elementos sem valor semântico descontinuados

Semântica de Marcações

  • Versões anteriores do HTML não haviam tags com semântica
  • Usavam div para todas as situações, junto com id e class
  • HTML5: Tags semânticas para indicar aos user-agents cada conteúdo da página

Metadados

  • Conjunto de informações à respeito da página
  • Invisível para aos usuários
  • Usada pelos user-agents em geral

Metadados

Header

  • Define um cabeçalho de uma página
  • Pode conter logo, títulos, campo de busca, menu de navegação, etc

Footer

  • Define um rodapé da página
  • Pode conter copyright, blocos de navegação, links relacionados, informações sobre o domínio, etc

Section

  • Similar à uma div
  • Ao contrário da div que divide qualquer conteúdo sem uma finalidade específica, a section divide diferentes seções do site
